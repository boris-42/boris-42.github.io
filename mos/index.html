What is OpenStack?

OpenStack is a cloud operating system that controls large pools of compute, storage, and networking resources throughout a datacenter, all managed through a dashboard that gives administrators control while empowering their users to provision resources through a web interface. Find more details here

More Tips For You

Murano - run your apps in cloud!

The Murano Project introduces an application catalog to OpenStack, enabling application developers and cloud administrators to publish various cloud-ready applications in a browsable categorized catalog. Cloud users -- including inexperienced ones -- can then use the catalog to compose reliable application environments with the push of a button. Amazing? Find even more!

Heat - create envs from templates

The Heat project introduces an orchestration service that allows you to spin up multiple instances, logical networks, and other cloud services in an automated fashion. Find more about heat here
